# üìä Statistical Data Science

This repository contains a collection of Jupyter notebooks covering key topics in **Statistical Data Science**.  
Each notebook includes theoretical explanations, practical examples, computations, and visualizations designed to help learners understand modern statistical concepts.

---

## üìò **Contents**

The repository currently includes the following notebooks:

| Notebook | Description |
|---------|-------------|
| **1. Statistical_Data_Intro_1.ipynb** | Introduction to statistics, data types, levels of measurement, and foundational concepts. |
| **2. 2_Descriptive_Statistics.ipynb** | Measures of central tendency, variability, distribution shape, and exploratory data analysis. |
| **3. 3_Probability_Theory.ipynb** | Core probability rules, random variables, expectation, and applications. |
| **4. Probabilities_distributions_4_&_5.ipynb** | Detailed exploration of probability distributions (discrete & continuous). |
| **5. 8_Hypothesis_Testing.ipynb** | Statistical inference, hypothesis tests, p-values, confidence intervals, and practical examples. |
| **6. 10_GLM.ipynb** | Generalized Linear Models (GLM): logistic regression, Poisson regression, and link functions. |
| **7. 11_Longitudinal_Data_Analysis.ipynb** | Statistical methods for longitudinal and repeated measures data; mixed models and correlation structures. |
